######平衡二叉树

1. 在出入操作中，只要将最小不平衡子树调整平衡，则其他祖先结点都会恢复平衡。（H+1->H+2->H+1)
2. 所有非叶子结点的平衡因子均为1， 此时满足以nh表示深度为h的平衡树中含有的最少结点数。 n0 = 0, n1= 1, n2 = 2, nh = nh-1 + nh-1 + 1。
(PS: n3 = 4, n4 = 7, n5 = 12, n6 = 20, n7 = 33)


######哈夫曼树

1. 哈夫曼树不一定是完全二叉树， 且树中一定没有度为1的结点。
2. 树中权值最小的两个结点一定是兄弟结点。
3. 树中任一非叶结点的权值一定不小于下一层任一结点的权值。
